⚡ Google Ads — Additional Setup Required

The Google Ads API is different from the standard Google OAuth you've already connected. Even with the API enabled in Google Cloud Console, you need three more things:

  1. A Developer Token — apply in your Google Ads account → Settings → API Center. Test tokens work for your own accounts.
  2. Your Customer ID — the 10-digit number (xxx-xxx-xxxx) in the top-right of Google Ads.
  3. A Manager Account (MCC) if accessing multiple accounts — or use the direct Customer ID if you have one account.

Add GOOGLE_ADS_DEVELOPER_TOKEN and GOOGLE_ADS_CUSTOMER_ID to your Render env vars, then reconnect Google (the adwords scope will now be included) and click Sync.

⚡ Meta Ads data requires your Ad Account ID in Connect Sources. Google Ads will load once the Developer Token + Customer ID are set in Render env vars.
